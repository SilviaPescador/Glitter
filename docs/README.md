## 📊 Referencia Rápida

### Puertos
- **Backend:** 3000
- **Frontend:** 8080
- **MongoDB:** 27017

### Comandos Principales

```bash
# Backend
cd Glitter-api
npm install && npm run init-db && npm start

# Frontend
cd Glitter-Vue
npm install && npm run serve

# MongoDB
net start MongoDB  # Windows
mongosh            # Conectar
```
