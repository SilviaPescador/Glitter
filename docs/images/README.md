# 📸 Imágenes de Documentación

Esta carpeta contiene las imágenes utilizadas en la documentación del proyecto.

---

## 📋 Imágenes Requeridas

### **landing-page.png**
- **Descripción:** Captura de pantalla de la página principal de Glitter
- **Ubicación:** `docs/images/landing-page.png`
- **Usado en:** `README.md`
- **Dimensiones recomendadas:** 1200x800px o similar
- **Formato:** PNG o JPG

**Para añadir la imagen:**
1. Guarda tu captura de pantalla como `landing-page.png`
2. Colócala en esta carpeta (`docs/images/`)
3. La imagen aparecerá automáticamente en el README

## 📏 Recomendaciones

### Tamaño
- **Ancho máximo:** 1200px
- **Formato:** PNG (mejor calidad) o JPG (menor tamaño)
- **Optimizar:** Usa herramientas como TinyPNG para reducir tamaño

### Contenido
- ✅ Muestra funcionalidades principales
- ✅ Interfaz limpia y clara
- ✅ Sin datos sensibles
- ✅ Representa bien el diseño
